# Günstige Hardware-Variante Für Basselefant (Mit Mikro + Monitor)

## Kurzfazit

Ein **Arduino Nano** ist dafür nicht geeignet:
- Kein nativer HDMI-Ausgang
- Zu wenig Rechenleistung und RAM für FFT + künstlerische Echtzeit-Visuals

## Sinnvolle Low-Cost-Lösung

**Raspberry Pi Zero 2 W** (oder Pi 4/5 bei mehr Leistung)

Benötigt:
1. Raspberry Pi Zero 2 W
2. USB-Mikrofon oder USB-Soundkarte mit Mikro-Eingang
3. mini-HDMI -> HDMI Kabel/Adapter
4. Für alte VGA-Monitore: aktiver HDMI -> VGA Adapter
5. Netzteil + microSD

## Kostenrahmen

Typisch ca. **50–80 EUR/USD**, abhängig von vorhandenem Zubehör.

## Warum das passt

- Linux + GPU + HDMI nativ
- Reicht für Audioanalyse und visualisierte Shader/Canvas-Renderings
- Sehr günstig im Vergleich zu Mini-PC-Lösungen
